A simple UI app for BeamNG which instructs the driver on when to upshift or downshift for economical driving. This could be helpful for someone who is learning to drive.

![Driving with an upshift indication](/screenshots/shiftup.png)

- The rules for the indicators are as follows:
    - Upshift: 2500RPM
    - Downshift 1500RPM
    - Does not illuminate when in first or top gear
    - Does not illuminate when below 7 MPH
